# The intended 90-second path

1. **0–10 seconds:** one identity—products for communities and systems that leave evidence.
2. **10–30 seconds:** RGIT Rozgar founder signal plus Kairo technical signal.
3. **30–55 seconds:** current public repository state, including failures.
4. **55–75 seconds:** one reproducible test and its boundary.
5. **75–90 seconds:** challenge interface, repositories, and contact.

The package intentionally excludes trophy walls, contribution snakes, visitor counters, fake terminals, self-awarded skill percentages, unsupported scale claims, and generated praise.
